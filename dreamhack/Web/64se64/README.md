# wargame : 64se64

- 문제에 들어가서 서버를 생성하여 웹해킹 URL로 접속했다.
- 페이지에 Welcome! 이라고 밖에 적혀있지 않아서 F12를 눌러 개발자 도구를 활용하여 페이지내용을 살폈다.

- 내용중에 “name=64se64_encoding” 이라는 태그가 있어서 value값을 복사하여 decoding 시켜 보았다.
