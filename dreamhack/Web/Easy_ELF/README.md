- 보면 byte_804A021~~ 이런식으로 비슷한 주소들이 이리저리 섞여 나열되있다. 만약 중간에 하나라도 거스르면 return 0으로 함수가 끝나게 되는 구조여서 모두 만족하여 마지막 return byte_804A023 == -35 에서 참을 만족하면 되는 문자열을 찾아야 한다.
